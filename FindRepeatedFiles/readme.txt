This program find repeated files in a folder.

The files can be considered repeated according to name, content or both (name and content)

To run is very simple:

java Main <folder> <option>

e.g.


To find files with same name:

java Main c:\music -n


To find files with same name and content:

java Main c:\music -cn



To find files with same content:

java Main c:\music -c
