This program Set Properties and rename mp3 file to Music name - Author

It is very simple to use.

You can change multiple files using a bat file.


Note: 


C:\Dados\JCD\MyTools\ChangeMusicProp>python ChangeMusicProp.py

Use python ChangeMusicProp.py Filename Artist Album Music

E.g. 

C:\Dados\JCD\MyTools\ChangeMusicProp>python ChangeMusicProp.py test.mp3 JCD "My Music" Lálálá
Processing:  test.mp3  -  JCD  -  My Music  -  Lálálá
